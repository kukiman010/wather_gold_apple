## Описание (Description)

Показывается в пустом чате **до** нажатия «Start». Лимит — **512 символов**.

```
Бот следит за ценами на товары в «Золотом Яблоке» (goldapple.ru).

Как пользоваться:
1. Отправьте ссылку на товар с сайта
2. Укажите пороговую цену в рублях
3. Получите уведомление, когда цена опустится ниже порога

Бот проверяет цены автоматически (по умолчанию — раз в час). Учитываются обычная цена, скидки и цена по карте лояльности.

Команды: /list — ваши товары, /cancel — отменить добавление.

Сервис для покупателей, не связан с магазином.
```

**Символов:** ~430

## Команды (Set Commands)

```
start - Начать работу с ботом
list - Список отслеживаемых товаров
cancel - Отменить добавление товара
```

## Privacy Policy (Privacy Mode)

- В BotFather включите **Privacy Mode: Disabled**, если бот должен видеть все сообщения в группах (для личного использования обычно не требуется).
- Бот хранит только `user_id` Telegram и данные об отслеживаемых товарах (ссылка, название, цены).
- Ссылка на политику конфиденциальности — при публикации в каталоге Telegram добавьте URL (при необходимости).

## Чеклист настройки в BotFather

| Шаг | Команда BotFather | Что указать |
|-----|-------------------|-------------|
| 1 | `/setname` | Имя из раздела выше |
| 2 | `/setdescription` | Полное описание |
| 3 | `/setabouttext` | Короткий текст «О себе» |
| 4 | `/setcommands` | Список команд |
| 5 | `/setuserpic` | Иконка 512×512 |
| 6 | `/token` | Скопировать в `.env` → `BOT_TOKEN` |
